About Cornugopia

Cornugopia is an open-source Go server designed for friendly matches— no login required. It supports both private games with friends and public games with custom rules.

View the source on GitHub.

Privacy Policy

We don’t use any third-party trackers or cookies. The only data stored on your device is a temporary player ID in localStorage, used to keep you identified across sessions.
